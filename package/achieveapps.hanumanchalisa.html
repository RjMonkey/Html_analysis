What information we collect

  • Reroute Outgoing Calls: This permission is used in specific Outgoing call to unhide the hidden application.
  • Read Phone Status & Identity: This permission allows to recognize an incoming / outgoing call and to switch between the User System interface and call interface or announces / shows the caller name while some one is calling, you will identify it without looking to your smart phone.
  • Read the contents of your USB storage : This permission allows to view app data(image, text, audio etc..) stored on their SD card.
  • Modify or delete the contents of your USB storage : This permission allows to delete app data(image, text, audio etc..) stored on their SD card.
  • Adjust your wallpaper size :This permission allows users to crop wallpaper to the desired size.
  • View network connections:This permission is used in settings and notification toolbar, in order to view network connections.
  • Full network access :This permission is used to access the device’s network for certain functions including receiving update notifications or accessing app classification labels.
  • Set wallpaper:This permission allows users to set a selected image as their wallpaper.
  • Prevent device from sleeping : This permission allows to prevent your device’s processor from sleeping. It keeps the app awake while doing praying process.
  • Receive data from Internet : Pocket uses your device’s internet connection to sync, download content from your list for offline access and display webpages you open. Pocket gives you a variety of settings to control what and when content is downloaded and synced.
